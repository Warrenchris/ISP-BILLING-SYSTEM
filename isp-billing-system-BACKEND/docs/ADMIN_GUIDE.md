# ISP Billing System - Admin Guide

This guide provides instructions for administrators on how to manage the ISP Billing System.

## 1. User Management

### 1.1 Viewing All Users

Admins can view a list of all registered users in the system.

### 1.2 Managing User Roles

Admins can assign and modify user roles (customer, admin, support) to control access levels.

### 1.3 User Account Actions

Admins can perform actions such as suspending or deactivating user accounts.

## 2. Data Plan Management

### 2.1 Creating Data Plans

Admins can create new data plans, specifying details such as name, description, data limit, price, and validity.

### 2.2 Updating Data Plans

Admins can modify existing data plans, including price changes, data limit adjustments, or activation/deactivation.

### 2.3 Deleting Data Plans

Admins can remove data plans from the system.

## 3. Subscription Management

### 3.1 Viewing All Subscriptions

Admins can view a comprehensive list of all active and inactive subscriptions.

### 3.2 Managing Subscriptions

Admins can update subscription statuses (e.g., activate, suspend, cancel) and modify other subscription details.

## 4. Payment Management

### 4.1 Viewing All Payments

Admins have access to a complete history of all payments made through the system.

### 4.2 M-Pesa Configuration

Admins can check the status of the M-Pesa integration and ensure all necessary configurations are in place.

### 4.3 Testing M-Pesa STK Push

In development environments, admins can initiate test M-Pesa STK Push requests to verify the integration.

## 5. Invoice Management

### 5.1 Generating Invoices

Admins can generate invoices for individual subscriptions or in bulk for multiple users.

### 5.2 Viewing All Invoices

Admins can view all invoices, filter by status, and track payment statuses.

### 5.3 Marking Invoices as Paid/Overdue

Admins can manually update the status of invoices, marking them as paid or overdue.

### 5.4 Sending Payment Reminders

Admins can send automated payment reminders to users with overdue invoices.

### 5.5 Billing Statistics

Admins can access billing statistics to monitor revenue and payment trends.

## 6. Data Usage Tracking

### 6.1 Viewing System-Wide Usage Statistics

Admins can view aggregated data usage statistics across all users.

### 6.2 Terminating Sessions

Admins have the ability to terminate active data usage sessions.

## Support

For any administrative issues or system support, please refer to the deployment guide or contact the system developers.
